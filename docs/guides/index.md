# User Guides

The guides are organized by workflow rather than by module.

```{toctree}
:maxdepth: 1

circuit-modeling
bbq-workflow
transition-fitting
point-picking
iterative-diagonalization
bbq-circuit-designer
```

Use them in this order if you are new to the project:

1. start with circuit modeling
2. move to BBQ if your inputs come from circuit matrices
3. use point picking and transition fitting for spectroscopy data
4. read the iterative diagonalization and theory pages for the modeling details
